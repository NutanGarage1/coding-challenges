# 🤝 Contributing to NutanGarage

Thank you for your interest in contributing! Every contribution helps make NutanGarage better for everyone. 🚀

---

## 📌 How to Contribute

### 1. Fork the Repo
Click the **Fork** button on the top right of any NutanGarage repo.

### 2. Clone Your Fork
```bash
git clone https://github.com/YOUR_USERNAME/REPO_NAME.git
cd REPO_NAME
```

### 3. Create a Branch
```bash
git checkout -b feature/your-contribution
```

### 4. Make Your Changes
- Follow the existing file structure and formatting
- Use clear, beginner-friendly language
- Add comments in code solutions where helpful

### 5. Commit & Push
```bash
git add .
git commit -m "Add: describe your change here"
git push origin feature/your-contribution
```

### 6. Open a Pull Request
Go to your fork on GitHub and click **New Pull Request**.

---

## ✅ Contribution Guidelines

### For Coding Challenges
- Follow the existing challenge template
- Include problem statement, approach, time & space complexity
- Provide solutions in as many languages as possible
- Add LeetCode / practice link at the bottom

### For Roadmaps
- Keep content beginner-friendly
- Include free resources where possible
- Mention timeline estimates

### For Resources
- Only add high-quality, well-maintained resources
- Use the correct legend tags (🆓 💰 🎓 📖 🎥 🌐 🏋️)
- Verify links are working before submitting

---

## 🚫 What NOT to Do
- Don't copy-paste entire articles or tutorials
- Don't add broken or spammy links
- Don't change formatting without discussion
